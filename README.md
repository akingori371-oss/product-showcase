# Chiwenga Tech Kenya

## Overview

Chiwenga Tech Kenya is a responsive frontend e-commerce website built with HTML, CSS, and Tailwind CSS. The project showcases a collection of technology products in a clean grid layout and includes the foundation for a shopping cart system.

This project was created to practice modern frontend web development concepts such as responsive layouts, semantic HTML, CSS styling, and basic user interface design.

## Features

* Responsive product showcase
* Six featured technology products
* Product images and descriptions
* Product pricing
* "Add to Cart" buttons for every product
* Sticky shopping cart sidebar
* Dark mode button (UI prepared for future JavaScript functionality)
* Modern layout built using Tailwind CSS
* Semantic HTML structure

## Technologies Used

* HTML5
* CSS3
* Tailwind CSS (CDN)
